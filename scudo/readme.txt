Lightweight GDPR compliance. For real. Cookie banner, script blocking, granular consent, Google Consent Mode v2, privacy policy wizard — all in 12KB.

== Description ==

**Scudo** makes your WordPress site GDPR compliant in 2 minutes.

A guided wizard asks who you are, what your site does, and which services you use. Scudo does the rest: creates Cookie Policy, Privacy Policy and Data Rights pages, activates the cookie banner and blocks all tracking scripts before consent.

= Why Scudo? =

* **12KB total** CSS + JS — the lightest cookie banner on the market
* **Zero dependencies** — vanilla JavaScript, no jQuery, no external libraries
* **Italian Garante Privacy compliant** — June 2021 guidelines
* **EU GDPR compliant** — Regulation (EU) 2016/679
* **Google Consent Mode v2** — built-in, Basic mode
* **2-minute setup** — guided wizard with autodetect

= Features =

* Cookie banner with equal-prominence Accept/Reject buttons (Italian Garante requirement)
* Close button (X) equals rejection (Garante requirement)
* Granular preferences panel (necessary, analytics, marketing, preferences)
* Preventive blocking of 30+ tracking services (Google Analytics, Facebook Pixel, YouTube, Maps, Hotjar, etc.)
* Descriptive placeholders for blocked content (videos, maps, social widgets)
* Consent logging in the database (legal proof for inspections)
* Configurable consent expiry (default 6 months as per Garante)
* Google Consent Mode v2 with all 7 parameters
* Automatic cookie scanning
* Built-in database of 30+ known cookies with descriptions
* `[scudo_cookie_table]` shortcode for Cookie Policy page
* Privacy Policy wizard compliant with GDPR Articles 13-14
* `[scudo_privacy_policy]` shortcode for dynamic rendering
* Automatic Google Fonts self-hosting (fixes the Munich court ruling 2022 issue)
* Automatic integration with Contact Form 7, WPForms, Gravity Forms
* Consent checkbox in WordPress comments
* GDPR data rights form (access, rectification, erasure, portability, objection)
* Integration with WordPress Privacy Tools (export/erase data)
* Email notifications for data rights requests
* CSV export of consent logs for audits
* Multilingual support (WPML, Polylang)
* Cookie preferences re-open widget
* Dark/light/custom banner theme
* WCAG 2.1 AA accessible (keyboard, screen reader, contrast, focus management)
* Cache plugin compatible (smart bypass when consent is already given)

== Installation ==

1. Upload the `scudo` folder to `/wp-content/plugins/`
2. Activate the plugin through the "Plugins" menu
3. Follow the guided setup wizard (appears automatically)
4. Alternatively, go to Settings → Scudo for manual configuration

= Quick Setup (Wizard) =

The wizard guides you in 5 steps:

1. **Who you are** — Your business details (pre-filled from site settings)
2. **What your site does** — Scudo auto-detects installed plugins
3. **Which services you use** — Scudo scans your site and pre-selects detected ones
4. **Choose the theme** — Dark or light
5. **Activate Scudo** — Automatically creates Cookie Policy, Privacy Policy, and Rights pages

= Available Shortcodes =

* `[scudo_cookie_table]` — Cookie table for Cookie Policy page
* `[scudo_privacy_policy]` — Privacy Policy generated by the wizard
* `[scudo_rights_form]` — Form to exercise GDPR data rights

== Frequently Asked Questions ==

= Does Scudo slow down my site? =

No. Scudo weighs 12KB total (CSS + JS) with no external dependencies. If the visitor has already given consent, the output buffer doesn't even activate — zero overhead.

= Does it work with caching plugins? =

Yes. Consent is managed entirely client-side (cookie + JavaScript). The output buffer deactivates automatically when consent has been given.

= Do I need to configure Google Consent Mode manually? =

No. Enable the option in settings and Scudo handles everything: sets defaults to "denied" and updates them after consent.

= Can I use Scudo with WPML or Polylang? =

Yes. Scudo automatically registers all banner strings for translation.

= Are Google Fonts blocked? =

Yes. You can also download them locally with one click from the Tools tab — so your site no longer transfers visitor IPs to Google.

= Is the plugin compliant with the Italian Garante Privacy? =

Scudo was designed following the Garante's June 10, 2021 guidelines. Equal-prominence buttons, X = rejection, no dark patterns, 6-month expiry, consent logging.

= What happens if I deactivate the plugin? =

Consent logs remain in the database (they are your legal proof). Pages created by the wizard stay published. You can reactivate the plugin at any time.

= What happens if I delete it completely? =

All plugin data is removed: options, database tables, and downloaded fonts.
